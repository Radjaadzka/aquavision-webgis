# 13 – ANALISIS FINANSIAL

## Estimasi Biaya Proyek

**Total estimasi biaya:** Rp 122.000.000 (Rp 122 juta)

Dasar perhitungan: Standar **Ikatan Nasional Konsultan Indonesia (INKINDO) Tahun 2024**, disesuaikan dengan indeks Nasional sebagai area kerja.

---

## Komponen Biaya

| Jenis Biaya | Keterangan |
|---|---|
| **CAPEX (Capital Expenditure)** | Modal awal/investasi untuk membangun sistem dari nol: Rp 122.000.000 |
| **OPEX (Operational Expenditure)** | Biaya rutin pemeliharaan tahunan: estimasi 10% dari CAPEX = Rp 12.200.000/tahun |

---

## Analisis Kelayakan Finansial (5 Tahun)

### 1. Benefit Cost Ratio (BCR)

**Rumus:**
```
BCR = Benefit / (CAPEX + OPEX)
```

**Perhitungan:**
```
BCR = (190.000.000 × 5) / [122.000.000 + (12.200.000 × 5)]
BCR = 950.000.000 / 183.000.000
BCR = 5,19
```

**Interpretasi:** BCR ≥ 1 → proyek dinyatakan layak (feasible). BCR = 5,19 berarti setiap Rp 1,00 yang diinvestasikan menghasilkan manfaat ekonomi sebesar **Rp 5,19** bagi masyarakat, sektor wisata, dan pemerintah desa. Nilai ini jauh melampaui ambang minimum → proyek dikategorikan **berdampak ekonomi tinggi**.

---

### 2. Net Present Value (NPV)

**Asumsi:** Tingkat diskonto (discount rate) r = 10%.

**NCF tahunan (Net Cash Flow):**
```
NCF_t = Benefit − OPEX = 190.000.000 − 12.200.000 = Rp 177.800.000
```

**Rumus NPV:**
```
NPV = Σ [NCF_t / (1 + r)^t] − CAPEX
```

**Perhitungan (3 tahun pertama):**
```
NPV = (177.800.000/1,1¹ + 177.800.000/1,1² + 177.800.000/1,1³) − 122.000.000
NPV = 442.659.655 − 122.000.000
NPV = Rp 320.659.655
```

**Interpretasi:** NPV positif → proyek menguntungkan secara ekonomis dari perspektif nilai sekarang.

---

### 3. Return on Investment (ROI)

**Rumus:**
```
ROI = Net Profit / (CAPEX + OPEX) × 100%
```

**Net Profit:**
```
Net Profit = (190.000.000 × 5) − 122.000.000 − (12.200.000 × 5)
Net Profit = 950.000.000 − 122.000.000 − 61.000.000
Net Profit = Rp 767.000.000
```

*(Catatan: Dokumen asli menunjukkan hasil ROI = 485,79%; terdapat ketidaksesuaian angka Net Profit dalam dokumen – lihat seksi Konflik Informasi di file 08.)*

**ROI = 485,79%** → Setelah menutup semua modal dan biaya operasional, sistem memberikan keuntungan tambahan sebesar ~4,86 kali lipat dari modal yang dikeluarkan.

---

## Estimasi Manfaat

**Manfaat tahunan yang diperkirakan:** Rp 190.000.000/tahun

Manfaat diperkirakan kepada tiga kelompok:
1. **Masyarakat** – efisiensi distribusi air, pengurangan konflik sosial.
2. **Pelaku Fasilitas Wisata** – perencanaan operasional dan pengembangan yang lebih efektif.
3. **Pemerintah Desa / BUMDes** – pengelolaan aset dan pengambilan keputusan berbasis data.
